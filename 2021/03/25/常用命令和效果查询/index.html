# 分类和标签

分类：categories

标签：tags

分类具有顺序性和层次性，类 A, 类 B 不等于类 B, 类 A；标签无顺序和层级性

# 为文章添加多个分类的方法

categories:

​ - [父，子 A]

​ - [父，子 B]

​ - [C]

如此，这篇文章同时包括三个分类： 子 A 和 子 B 分别都是父的子分类，同时 C 是一个没有子分类的分类。
