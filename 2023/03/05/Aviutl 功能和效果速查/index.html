# 常用快捷键
  • S: 分割物件
  • L: 隐藏 / 显示当前图层
  • p: 在当前位置打上关键帧
  • T：更改物件长度（精确时间）
  • G：显示 / 隐藏辅助线
  • E：添加效果
  • V：调出修改可视窗口占比控件
  • M：最后物件位置为最后一帧
  • Ctrl+L：锁定当前图层
  • Ctrl+N：创建新项目
  • Ctrl+O: 打开存档
  • Ctrl+T：均匀分布物件上的中间点
  • Ctrl+D: 向下生成物件副本
  • Ctrl+Shift+S：导出 MP4
  • Ctrl+Shift+A：导出 AVI
# 文本跑马灯

不分行输入后启用自动滚动（左对齐模式或竖排版上对齐）

# 文本报幕

分行输入后启用自动滚动（只限上对齐）

# 字号强调

<s80> 想改变字号的内容 < s>，s 表示字号

# 颜色强调

<#fb7299> 想改变的颜色的内容 <#> #表示颜色

# 文字停顿

<w0.5> 嘿！<w > 出去走走吗？w 表示停顿的时长，单位是秒，精度为小数点后一位，可与显示速度配合使用

# 定时显示字幕
  • 大家好 <c1.0>
  • 欢迎收看 xx 看世界 <c1.0>
  • 我是 xx <c1.0>

c 表示字幕停顿的时长，单位秒，精度点后一位，可与显示速度配合使用

# 文字路径

为文本物件添加两个中点（P）右键设置按时间分配中点，设置每个关键帧的位置，勾选独立每个文字，启用在路径上显示，可配合自动滚动使用

# 视频倒放

设置视频播放速度为负，-100 为常速倒放

# 物件倒放

在物件上方添加与物件长度相同的时间控制（媒体物件），设置剪辑控制的位置由 100->0

# 加减速运动

在物件靠前端处添加中点，设置运动类型为加减速运动

# 固定量运动（自定义运动量）

根据设定的固定运动量进行运动

# 反复运动（往返运动）

在设置完运动类型后，可以在底部设置运动时间间隔（单位：帧）

# 随机运动

设置完运动类型后，可以在底部设置随机运动的时间间隔（单位：帧）

# 分组

按住 Ctrl 键同时选中多个物件右键其中一个物件设置分组，绑定物件的移动的相对位置。再次右键可以取消分组

# 组控制

同时控制多个物件，右键 -> 添加媒体物件 -> 组控制，设置目标图层数量后可对下方物件进行三轴移动，三轴旋转，缩放率以及特效的同步。

  • 组控制可以通过启用受上方组控制进行嵌套，
  • 组控制无法作用于已经分组的物件
# 自定义物件 - 雪

右键创建自定义物件，选择雪，除常规参数外，进入设置选项可以设置一些高级参数

  • 纵深：雪花的透视深度，数值越大则大小差异越明显
  • 单一化：将所有雪花合成整体来控制
# 自定义物件 - 显示音频波形

右键创建自定义物件，选择显示音频波形，除常规参数外，还有一些高级设置可供调整

  • 横 / 纵向分辨率：调整波形的可视化精度
  • 横 / 纵向范围：调整音频波形之间的空隙距离
  • 镜像：设置为 0 不启用镜像，设置为 1 启用镜像

可与极坐标变换配合使用

# 网格显示

右键勾选网格显示（XY 轴），再次右键进入网格设置，在显示网格间隔（XY）轴中设置高度和宽度（单位：像素）

汉化版 bug：只显示 Y 轴网格不显示 X 轴

# 裁剪遮罩

对物件添加裁剪遮罩效果，可以从上下左右四个方向对物件进行裁剪，勾选改变中心位置可以保持当前物件的中点为中心

可以实现的效果：进度条，卡拉 OK 歌词，卷轴展开

# 自定义物件 - 计数器

右键自定义物件，选择计数器。高级设置中，装饰类型可设置为 1,2,3,4，其中 1,2 为阴影样式，3,4 为描边样式

# 描边

为物件添加描边效果，除常规效果外，可以点击图片文件模式

选择图片对描边部分进行填充（分辨率过大的图片无法使用），当模糊数值大于 70 会接近外发光效果

# 单独滤镜

右键添加效果物件，选择单独滤镜。在此物件上添加的效果只会在局部可见图形区域内显示，选择遮罩类型可以改变遮罩的形状，勾选遮罩反转可以反转效果的显示区域

# 场景嵌套

单击左上角主合成可以进入各个预合成。右键则可以设置预合成的画面大小和开启 alpha 通道（主合成无法开启 alpha 通道）。

在空白处右键添加媒体物件 -> 预合成，可以将预合成导入到当前合成，通过预合成选择可以选择其他预合成。

预合成开启 alpha 通道后，添加的全局效果物件将会失效

# 遮罩

为物件添遮罩效果后，遮罩范围之外的内容将会被隐藏，通过遮罩类型可以跟换遮罩形状，勾选适合原始大小则会让遮罩自动匹配物件大小

可以实现的效果：探照灯，朦胧睁眼等，配合水墨素材还可以实现点墨，划墨转场

# 色彩偏移（RGB 分离）：

为物件添加色彩偏移效果，将偏移度属性设置为随机运动（范围 0-3），周期设置为 1，可以实现电视信号的电子干扰效果

# 抖动（振动）

为物件添加抖动效果，通过改变周期改变抖动的频率，反复振动会在原本的抖动之上再添加一个抖动

可以实现的效果：镜头晃动，与色彩偏移配合使用

# 剪贴蒙版

图形（或带透明背景的图片）叠置在目标图片或视频上方，右键目标图片，选择用上方物件裁剪，即可使图片裁剪成上方图形形状，通过调整图形大小可以调整蒙版大小

# 光栅

为物件添加光栅效果，宽度越大扭曲越明显，高度越小栅格越密集。周期越居中运动频率越小。勾选随机振幅可以产生连续不规则的光栅边界

可以实现的效果：旗帜飘扬，果冻抖动

# 镜像

为物件添加镜像效果。透明度调节镜像整体透明度，衰减调节镜像的远距离渐隐效果，调整镜面可以调整镜像与本体的距离。反射改变镜像的对称方向，改变中心位置将使原物件中心移动到添加效果后的整体中心位置。

自定义物件 - 涟漪效果（随机环粒子）：右键创建自定义物件，选择随机环粒子，通过改变时间调整单个图形的停留时间，间隔则是调整下一个图形出现的时间间隔。除常规参数外，还有一些高级设置可供调整

  • 种类 & 颜色：改变图形的形状和颜色
  • 彩色选项：为图形赋予随机色彩（此时原先设定的颜色会失效）
  • 随机角度：开关图形的随机旋转
  • 虚拟 buff 大小：调整图形整体的显示范围
# 自定义物件 - 朦胧光晕（暗角）

右键创建自定义物件，在下拉菜单栏中选择朦胧光晕，可调整属性如下：

  • 光亮：调节边缘光圈亮度
  • 半径：调整光圈直径大小
  • 边缘：修改效果模糊范围
  • 颜色偏移：勾选后为边缘添加色彩偏移效果

此外，设置选项还提供了一些高级属性

  • 光照强度：数值越大，光线越强
  • 色彩偏移量：数值越大：色彩偏移效果越明显
# 动画效果 - 平铺（图形拼贴）

为物件添加动画效果，在下拉列表中选择 图像拼贴 2@动画效果 1，除了一些常规属性外，设置选项还提供一些高级属性

  • Z 间隔：调整 Z 轴的平铺间隔
  • Z 个数：调整 Z 轴的平铺数量
  • XY 轴级差：使 XY 轴素材相互错开排列
  • ZY 轴级差：使 ZY 轴素材相互错开排列
  • 个别基准：使得效果生成物件独立，不勾选则视为一个整体
# 动画效果 - 破碎

为物件添加动画效果，在下拉列表中选择破碎，该效果有如下属性可供调整：

  • 开始时间：对象停留设置的时间再破碎
  • 中心 X,Y,Z：分别调整三轴上物件开始碎裂的位置

此外，设置选项还提供了一些高级属性

  • 碎片大小：修改每一片碎片的尺寸
  • 速度：调整碎片掉落时的速度
  • 重力：改变碎片掉落时的引力大小，为 0 则表现为失重，为负则反向上浮
  • 时间差：调整碎裂的延迟时间
  • 距离影响：调整碎片扩散范围
  • 随机旋转：调整每个碎片随机自转的速度
  • 随机方向：调整每个碎片随机扩散的范围
  • 播放速度：破碎动画的播放速度，为负则表现为倒放，且必须预先设置好开始时间
# 模糊

通过虚化背景强调主题；运动状态下模拟调焦，属性如下

  • 范围：调整模糊范围
  • 纵横比：负纵正横
  • 光照强度：使原本亮度较高区域更亮
  • 固定大小：勾选后只会在素材范围内模糊而不向外扩散
# 阴影

表现物件的层次关系，光影和空间感，美化视频的细节表现

为物件添加阴影效果，可调整属性如下

  • X，Y：调节阴影方向，表现光照方向
  • 浓度：调节阴影强度，表现光源的明暗
  • 扩散度：调整阴影扩散范围，表现光源的集散程度
  • 用另一个物件渲染阴影：将阴影作为独立物件，之后添加其他特性不会作用于阴影
  • 图片文件模式：使用指定图片填充阴影
# 动态模糊

使运动中的物体出现较明显的运动痕迹，增加流畅感

为物件添加运动模糊效果，可调整属性如下

  • 间隔：调整模糊痕迹的整体长度
  • 分辨率：运动分解的残影数值
  • 残影：取消勾选状态下可以表现出物件因高速运动以致物件本身也变得模糊的效果
  • 离屏渲染：默认勾选，取消勾选则残影不可见
  • 导出时提高分辨率：导出视频时优化效果精度
# 擦除

多用于转场，为物件添加擦除效果后在下拉栏里选择一种擦除预设，可调整属性如下：

  • IN，OUT：转场过程的持续时长
  • 模糊：擦除边缘的模糊程度
  • 反转（进场 / 出场）：翻转擦除的上下，左右，内外
# 发光

使画面更加美观和引人注目，增添梦幻感

为物件添加发光效果，可调整属性如下：

  • 强度：调整发光强度，不宜过高
  • 扩散度：调整发光效果的扩散范围
  • 阈值：调节光的作用范围
  • 模糊：调整光的模糊程度
  • 形状下拉菜单：可选择六种不同的发光形状
  • 仅渲染发光部分：勾选后素材只显示光线部分
# 剪影

为物件添加单色化效果，取消勾选亮度保持，设置颜色为黑色即完成

# 提取边缘

自动提取图片的线条边缘，做成线稿效果

为物件添加提取边缘效果，可调整的属性如下：

  • 强度：调整线条的清晰度
  • 阈值：调整明暗判定的临界值，改变线条提取的作用范围
  • 提取亮度边缘：勾选后可以使线条的明暗对比更加明确
  • 提取透明度边缘：只提取图片的边缘而忽略中间的明暗信息
# 添加特效（针对局部，上层和全局）

为局部添加：右键添加效果物件，选择单独滤镜放置在物件下方，调整选取框范围再点击 + 添加效果

为单层添加：右键添加媒体物件 - 添加滤镜效果，选择可选的效果生成效果物件后置于目标层下方

为全局添加：右键添加效果物件，选择可选的效果生成效果物件后置于目标层下方

# 渐变：

常用于背景色和文字，使画面颜色富于变化

为物件添加渐变效果，可调整的属性如下：

  • 强度：调整渐变对素材的干涉强度
  • 中心 X,Y：调整渐变图形的位置
  • 角度：调整渐变的角度
  • 宽：调节渐变图形的范围大小
  • 渐变性状下拉栏：更改渐变图形形状
  • 开始 / 结束颜色：选择渐变的两种颜色
# 反转（翻转）

为物件添加基础效果 - 反转，可以通过复选框选择反转模式

  • 上下，左右翻转：
  • 亮度翻转：反转暗色和亮色
  • 色相反转：颜色反转为互补色
  • 透明度反转：反转物件的透明区域和实体区域

物件 / 效果预设（exa 文件）：

完整保存单个素材内容及属性

物件预设（包含特效预设）

  • 设置方法：将调整好参数和特效的物件右键 - 创建预设（.exa），需要复用时直接拖入工程或者右键 - 添加媒体物件 - 预设即可

特效预设 **：** 为不同物件添加相同参数的特效

​ 设置方法：右键效果面板 - 保存设置 - 设置另存为 - 输入文件名和保存文件夹后，即可在效果竖列指定的文件夹里快速调用预设

无论是物件预设还是效果预设，其文件夹中都不能再次嵌套子文件夹，但可以创建多个物件或效果预设文件夹

若不输入文件夹名则 exa 文件会直接生成在软件根目录

在建立预设后将原素材删除将导致预设不可用

# 导出物件（exo 文件）

完整保存多个素材的内容及属性

适用场景：将某个可复用的组合效果物件导出为 exo，省去重新制作

设置方法：制作好一个包含多个素材的效果后右键空白处 - 文件 - 导出物件，命名好后保存，需要复用时直接拖入软件或者右键 - 导入物件即可（导入后不会合成一个物件）

exo 文件的优异性：相比于.aup，多个相同的 exo 文件可以同时导入并在一个工程内共存

* 为防止素材缺失，尽量将素材和 exo 文件放置于同一个文件夹内

# 特效初始值（exc 文件）

将一组参数设置某个特效的初始值（跨工程有效）

适用场景：将某个特效常用的参数效果设为特效预设，省去初始值调试

设置方法：右键 - 保存设置 - 设置为初始值

取消方法：在软件目录删除对应的.exc 文件

# 音画分离

导入视频后打开音频属性，取消勾选配合视频文件后右键音频执行取消组操作即可分离音画，通过设置浏览文件还可以关联其他媒体的音频

若不取消配合视频文件直接取消编组会导致音频失效

非整数帧率或非主流格式编码都可能导致音画不同步，需要先进行转码操作后再导入

# 动画效果 - 万花筒

以三角形作为单元图形在原始素材中进行局部选取，为物件添加动画效果，在下拉菜单选择万花筒后，可调整的属性如下：

  • 中心 X,Y：修改三角形选取位置的坐标轴
  • 长度：调整基础三角形的选取范围大小
  • 旋转：调整三角形的旋转角度
  • 显示位置确认：勾选后显示单元三角形的选取范围并隐藏效果

此外，设置选项还提供了一些高级属性

  • 角度（偶数）：调整万花筒中心顶点对应边数，只能设置为偶数
  • 重复计数：修改万花筒向外扩展的重复层数
  • 固定大小：修改效果显示大小
  • 圆形遮罩：使效果的多边形边框变成圆形
  • 旋转同步：勾选后在调整基础属性的旋转时，外框也跟着旋转，不勾选则只有内部旋转
  • 抗锯齿：使边缘变得平滑
# 物件分割

为物件添加物件分割效果，可调整横 / 竖分割数改变分割块数

点击 + 边上的转换选项，更改变换模式为扩展变化，属性扩展后如下

  • 缩放率：调节每一个方块的大小
  • 纵横比：调节每个方块的纵横比
  • X,Y,Z 轴旋转：调节方块在三轴上的旋转方向

物件分割后每个部分独立，因此之后添加其他效果会作用在每个部分从而实现一些特殊的效果

# BPM 辅助线

在空白处右键并勾选网格显示（bpm），右键网格设置进行调整，参考帧数可以微调辅助线的起始帧数来对上音频节奏

# 色彩修正

对素材进行色彩调整，为物件添加色彩修正效果，可调整的属性如下

  • 亮度：调节画面的亮度
  • 对比度：调节画面的明暗对比度
  • 色相：调节画面的色相
  • 发光度：调节画面的照度
  • 色度：调节画面的饱和度

亮度和照度的区别：照度指光源照在物体上的强弱，亮度则是物体反射光到人眼后的强度

# 特定色域转换

对画面的特定部分进行颜色修改

为物件添加特定色域转换效果，单击获取转换前的颜色初步选定转换范围（黑色遮罩部分），再调整以下三种属性进一步精确调整范围

  • 色相范围：通过色相调整选择范围
  • 色度范围：通过色彩饱和度调整选择范围
  • 边缘修正：调整选取范围的边缘

在确定调整范围后，单击获取转换后的颜色选择画面中的颜色替换

若想调整为画面中没有的颜色，可以添加一个色环到画面中

# 动画效果 - 颤动

为物件添加动画效果 - 在下拉栏里选择颤动，可调整属性如下

  • 振幅：调节颤动的幅度
  • 角度：调节颤动的角度
  • 间隔：调节颤动的频率，参数越大，颤动越慢，反之越快

# 摄像机

控制画面视角，营造立体感和空间感

创建摄像机：右键 - 添加媒体物件 - 摄像机控制

创建 3D 物件：右键 - 添加媒体物件 - 自定义物件，在下拉菜单中选择带边立方体 - 摄像机，打开高级属性框启用扩展变换，此时属性框左上角摄像机标识被点亮，说明该该物体视角能为摄像机所控制。

有关摄像机的若干操作

  • 选中摄像机控制物件后可以在播放窗口用鼠标拖动视角
  • 按住 Ctrl 的同时拖动右键可以调整摄像机的远近
  • 通过对摄像机控制物件添加关键帧来实现摄像机运动，添加关键帧后，包含起始帧和终止帧的所有关键帧会以方框形式显示在播放窗口上方

画面右下角的控件

  • 摄像机：切换到摄像机视角
  • 编辑：切换到自由视角
  • 前，后，左，右，上，下：正方体的六面视图，会自动切换到编辑视角（默认的正 / 前面为物体没有进行空间变化时的一面）

摄像机控制的属性如下：

  • X,Y,Z：改变摄像机的空间位置
  • 目标 X,Y,Z：不改变摄像机位置而调节摄像机的焦点位置（红线：摄像机对焦方向及焦点位置）
  • 目标层：自动锁定相应图层里的物体中心作为焦点
  • 倾斜：调整摄像机的拍摄角度
  • 深度模糊：为画面添加景深模糊（图形预览简化显示为开启状态则此效果在预览窗不显示）
  • 视角：调节摄像机的视角，数值越大，视角越广，可摄范围越大
  • 启用 Z 缓存 / 渲染阴影：物件按照空间位置关系在摄像机前显示，取消勾选则按照图层顺序显示
  • 目标图层数量：设置摄像机效果控制的层数避免影响其他图层

空间网格：直观显示和定位空间位置，辅助三维效果制作。右键 - 网格设置 - 网格间隔（摄像机控制），调整好宽和数量后右键勾选启用网格显示（摄像机控制）

添加立体阴影的两种方式：

  1. 在摄像机控制物件属性栏内通过右上角 + 添加阴影（摄像机控制）
  2. 右键 - 添加媒体物件 - 摄像机控制 - 阴影（摄像机控制）- 将生成的物件置入摄像机可控层内生效（优异：可以方便控制阴影出现的时间）

空间阴影的可调节参数如下：

  • 光源 X,Y,Z：调整光源位置（但光源本身不显示）
  • 浓度：节阴影强度，表现光源的明暗
  • 精度：调节阴影的清晰度

  • 间隔：摄像机抖动的频率，数值越小抖动越频繁
# 物件随机摆动（图形碎片）

填充画面，丰富层次

右键添加媒体物件 - 自定义物件，在下拉菜单栏中选择物件随机摆动（相机控制），可调整参数如下：

  • 数：碎片的数量
  • 范围：碎片的范围，范围越小碎片越密集
  • 大小：单个碎片的大小
  • 形状：改变碎片的形状

此外，设置选项还提供了一些高级属性

  • 颜色：碎片的颜色
  • 旋转速度：调节碎片旋转的快慢
  • 随机颜色：为碎片随机上色，但是灰度相同

利用阴影（摄像机控制）使碎片的颜色发生深浅变化

# 综合运用 - 星轨效果

右键添加媒体物件 - 自定义物件，在下拉菜单栏中选择扇形，可调整参数如下

  • 中心角：调节弧长大小，数值越大，弧长越大，形成的扇形面积越大
  • 中心基准：扇形的缺口方向
  • 大小：调整扇形整体大小
  • 线宽：调节扇形线宽

减小线宽，并为中心角属性添加 0-360 的运动，添加动画效果，在下拉菜单栏中选择随机分布 @动画效果 1，在设置中将 X,Y,Z 轴范围 [%] 分别设置为 0，,0,100，调整个数属性适中即可

动画效果 - 蝶：为物件添加动画效果，在下拉菜单栏中选择蝶 @动画效果 2，可调整参数如下

  • 振翅：调节蝴蝶振翅频率
  • 角度：调节翅膀振动幅度
  • 上下：调节上下摆动幅度
  • 羽（手动）：调节翅膀夹角偏差值（数值越大，上方夹角越小，下方夹角越大）
# 动画效果 - 模糊入场

为物件（文字需启用独立每个文字）添加动画效果，在下拉菜单栏中选择 TA - 模糊入场 @TA，可调整参数如下：

  • 时间 [s]：调整文单个字入场所需时间，正数入场，负数出场
  • 间隔 [s]：调整文字入场间隔
  • 范围：调节入场模糊范围，数值越大，模糊程度越高
  • 出场顺序：改变文字入场顺序，共有六种模式

此外，设置选项还提供了一些高级属性

  • feed：勾选后为进出场添加淡入淡出效果
  • 纵横比（-100~）：调整模糊方向，负纵向，正横向
  • 缩放率：调整效果整体缩放

叠加 TA - 标准登场 @TA，在高级设置中将 Y 距离属性设置为 200，为文字添加掉落效果（需将间隔和出场顺序设置为相同数值）

# 色键

快速去除素材中不需要的部分，常用于 GB,BB 素材

为物件添加色键效果，有如下属性可供调整

  • 获取主色调：点击预览窗口中欲去除的颜色进行取样
  • 亮度范围：调节取样点的亮度调节选择范围，数值越大，范围越大
  • 色差范围：通过颜色调节选择范围
  • 边缘修正：扩大选取范围的边缘
# 综合运用 - 不规则四边形蒙版

添加媒体物件，图形物件 - 四边形（默认白色），添加动画效果 - 简易变形，拖动顶点至合适形状后，右键欲裁剪的物件，执行用上方物件裁切，调整位置。将四边形的合成模式改成相乘，去除遮罩自身白色部分

漫画分镜框的制作

# 透明通道导出

文件 - 插件渲染 - 扩展剪辑 AVI/BMP 导出（RGBA），再次导入时，只需勾选读取 Alpha 通道即可使用

推荐的初始设置：视频压缩 - 压缩程序选择 Lagarith Lossless Codec，再进入设置，选择 Mode 为 RGBA，确认后保存

# 自定义物件 - 贝塞尔曲线（运动变化曲线）

用于调节物件运动过程中位移随时间变化的曲线形状，实现运动中更加复杂的速度变化

右键添加媒体物件 - 自定义物件，在下拉菜单栏中选择贝塞尔曲线 T，可调整参数如下

  • 轨道序号：曲线对应的编号，一个画面用到不同贝塞尔曲线时需将运动 - 设置运动帧间隔的数值和轨道序号对应
  • 轨道指定：修改物体的运动轨迹，共有四种模式
  • 图标大小：调节曲线大小，但调整后曲线形状也会产生变化
  • 显示图标：勾选后隐藏曲线以便影响画面
# 动画效果 - 球面化（球体分解）

为物件添加动画效果（同为动画效果的球体（摄像机控制）是其简化版），在下拉菜单栏中选择球体分解 - 摄像机 @动画效果 1，可调整参数如下：

  • 屈曲率：调节弯曲程度，数值越小越接近球体，反之接近平面
  • 中心 X,Y：调节图形开始扭曲的中心点位置（不建议通过对此属性添加动画实现转动，推荐通过扩展变化的三轴旋转实现）
  • 反转：勾选后翻转内外弯曲方向

Gitch 效果（振动 plus）：添加媒体物件 - 文本物件，键入内容后勾选独立每个文字并添加物件分割效果，将横分割数调到最小，添加动画效果，在下拉菜单栏中选择振动 Plus@动画效果 1，为振幅属性添加瞬间运动，添加 5 个关键帧并间隔设置这些关键帧的数值为 0 和 100，减小周期数值以加快抖动频率，添加色彩偏移效果，也添加瞬间移动以增强效果

# png 序列的导入

方法 1：文件 - AVI 文件操作 - AVI 文件合并 - 追加序列文件夹选择序列所在的文件夹后选中第一张图片打开，所有序列都会被导入（最多可连续识别 32 张），点击浏览设置输出文件夹，确认即可生成无损 AVI 视频文件，拖入软件即可使用

方法 2：文件 - 首选项 - 读取插件优先级 - 将 JPEG/PNG File Reader 上移置顶，如此视频导入格式就被修改为图片，新建媒体物件 - 视频物件，通过浏览文件选择序列所在的文件夹，选择序列的第一张，所有序列（无限制）都将被导入为视频（导入后务必要将读取插件优先级复原）
