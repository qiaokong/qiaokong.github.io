# 均衡器（Equalizer）

# 概念

# 在学习均衡器之前

# 声音的频谱成分
  • 基波（Fundamental）：一个声音最低的频率，它决定了声音的音高
  • 谐波（Harmonics）：频率为基波频率整数倍的成分
  • 泛音（Overtones）：频率不一定是基波频率整数倍的成分
  • 共振峰（Formants）：由物理共振产生的频率，不会随着音调变化而变化
# 频段的划分
  1. 低频（low）：20-250Hz
  2. 中低频（low-mid）：250-2kHz
  3. 中高频（high-mid）：2k-6kHz
  4. 高频（highs）：6k-20kHz
# 声音的性质
  1. 软：力度小，厚重，模糊，远，松散，慵懒
  2. 弹：介于软硬之间
  3. 硬：清晰，干净，迅猛，冲击力，近，结实
# 各频段的核心频率点
  1. 超低频：50Hz，越提升，声音越软
  2. 低频：150Hz，越提升，越硬，力度越大
  3. 中低频：450Hz，提升则温暖，模糊，不提升清晰度好
  4. 中频：1800Hz，提升声音进一步变硬
  5. 中高频：3500Hz，提升进一步变硬，清晰，同时带来刺耳感，不提升温暖模糊
  6. 高频：6500Hz，提升带来军鼓和镲的颗粒感和清脆质感
  7. 超高频：10000Hz，提升突出细节
  8. 极高频：14kHz，毛躁和不安感，人耳难以捕捉
  9. 空气声：16KHz+，提升则甜美温暖且清晰度下降

硬度与频率的线性关系：硬度会随着频率的增加而增加，直到 2000-3000Hz 达到最硬，之后随频率增加硬度减小

人耳的听阈：20-20k（单位：KHz）

# 均衡器的作用

  • 实现频谱平衡
  • 塑造乐器表现力
  • 实现乐器分离
  • 提高乐器清晰度
  • 塑造乐器的深度，前后位置

# 均衡器参数

  1. 频率（Frequency）：控制调整中心
  2. Q 值 / 带宽（Bandwidth）：调整影响范围
  3. 增益（Gain）：增益或衰减声音
  4. 斜率（Slope）：均衡曲线倾斜的程度，单位：dB/Octave（倍频程），数值越大，斜率越大

# 均衡曲线的基础形状

  • 参量式（bell）：以所选频点为中心进行增益或衰减
  • 搁架式（shelf）：对所选频点一侧的信号产生整体影响
  • 通过式（cut）：切除不需要的声音
  • 带阻（band stop）：以所选频点为中心，对带宽内所有声音进行切除
  • 带通（band pass）：以所选频点为中心，只保留设定频宽内的声音
  • 阴阳型（tilt）：以频点为中心同时反向提升和衰减两侧音量，用于总线处理

# 均衡器的种类

  • 参量式均衡器：可调整频点，带宽，增益

  • 图示均衡器：由众多小型推子构成，Q 值固定

  • 动态均衡器：均衡量随每一个频段信号的增益幅度而变化，某频段信号越大，增益或衰减的幅度越大，常用于人声，底鼓，配合高频搁架控制

# 使用均衡器的基本原则

  1. 不要生搬硬套预设 / 模板
  2. 宽提升
  3. 窄衰减：修复问题，如减小谐振
  4. 适度调整
  5. 整体考虑

# 实操

# 底鼓的均衡

20-50Hz：低频的饱满程度，缺少指向性；强硬的底鼓可以切掉该频段

60-150Hz：最重要的频段，其中真鼓 60-80Hz，力量感，需要与 bass 协调

200-350Hz：boxy 的声音，一般会进行衰减来为 bass，军鼓等乐器让出空间

500-1500Hz：提升底鼓踩锤的打击感（click）

# 军鼓的均衡

100Hz 以下：一般做低切处理

150-300Hz：军鼓的力量和打击感所在

500-1000Hz：适当衰减为其他乐器留出空间

1.5k-3kHz：提升此频段能突出响弦，增加亮度和穿透力

# Bass 的均衡

40Hz 以下：多数情况做低切处理

40-100Hz：最重要的频段，可以提升低频的饱满感，但是要与底鼓协调

100-250Hz：提升清晰度和律动（突出演奏感提升这一区域，可选）

300-600Hz：boxy 的声音，可以适当减少

800-1.2kHz：琴弦，品丝的声音，提升可以提升 bass 存在感

# 人声的均衡

80-100Hz 以下：基本做低切处理

100-200Hz：胸腔共鸣所在，提升厚度和磁性

300-600Hz: 鼻音，沉闷感，削减可以提升通透感

1.5k-3kHz：最重要的频段，提升人声辨识度和清晰度，但要注意控制齿音

7k-9kHz：削减以控制齿音

10kHz 以上：空气感，没有指向性，适当提高增亮，减少增暖

# 原声吉他的均衡

100Hz 以下：做低切处理

100-150Hz：演奏内容为旋律则保护，扫弦则控制

200-500Hz：提升突出箱体声音，提升木头的共振感

2k-5kHz：主要高频区，提升清晰度和通透感

5-10kHz：提升拨片扫弦的打击感

# 电吉他的均衡

100Hz 以下：做低切处理

100-250Hz：节奏吉他的力量感和打击感所在

250-400Hz：衰减以避免谐波堆积，提高清晰度并为其他乐器留出空间

15k-3kHz：最重要的频段，失真吉他一般控制该部分，提升清音音色的清晰度和穿透力

2-5kHz：主要高频区

10kHz 以上：基本可以削减以消除刺耳感

# 钢琴的均衡

100Hz 以下：做低切处理

100-200Hz：增加低频的饱满感，敲击琴键声所在

300-700Hz：削减该频段以提高通透感

1.5k-7kHz：最重要的频段

10k 以上：琴锤击弦音所在，提升轻快感

# 常需要处理的频段

  • 400Hz：谐振，谐波堆积
  • 3000-4000Hz
  • 8000-10000Hz：人声齿音，镲的呲呲声

# 均衡的侧链功能

侧链：用另一个音轨的信号驱动当前轨的插件

# 压缩器（Compressor）

# 概念

# 在学习压缩器之前

# 动态范围（Dynamic range）

一个系统所能容纳的最小和最大声音之间的差值。信号的电平起伏变化。单位为分贝 (dB) 大的动态可以增强表现力，小的动态能让音量趋于一致（用于单个乐器）

宏观动态：一段信号的电平变化状态，变化时间超过单一音符长度，如主歌与副歌之间的电平变化

微观动态：每一个音符随乐器演奏和发声规律产生的电平变化，可以通过动态包络表示这种变化

# 动态包络（Dynamic envelope）

又称 ADSR 曲线，由四个过程组成

  • 启动 Attack：信号从无到有直到最高点的时间
  • 衰减 Decay：从最高点掉落衰减的时间
  • 维持 Sustain：声音延续的时间
  • 释放 Release：声音从有到无直到消失的时间

# 压缩器的作用

  • 动态控制：通过设置合适的阈值和压缩比控制声音的动态，对动态范围进行压缩
  • 声音塑形：通过启动时间和释放时间改变音头和音尾从而改变声音特性
  • 音染色彩：模拟硬件压缩设备带来的谐波和音染，优化声音

# 压缩器的参数

基础参数

  • 阈值（Threshold）：压缩的触发界线，阈值以上的信号会触发压缩，其高低会决定压缩的幅度
  • 比例（Ratio）：信号超过阈值后，动态降低的比例，如 8:1 则每超过阈值 8dB, 动态降低 7dB
  • 拐点（Knee）：设定音量接近临界值时的压缩强度，值越低压缩越硬，响度越大；反之越平缓，过渡越自然
  • 启动时间（Attack）：信号超过阈值，压缩器等待的时间
    • 快速：＜5ms，用于压制音头，压缩动态
    • 中速：5-20ms
    • 慢速：＞30ms，强调音头，保持声音的特性和细节
  • 释放时间（Release）：声音恢复到未压缩状态所需要的时间（释放需要在两个主要拍子之间完成）
    • 快速：＜50ms，增加整体响度，营造 pumping 效果
    • 中速：50-100ms
    • 慢速：＞100ms，保持声音平稳自然，控制整体动态

扩展参数

  • 范围（Range）：设置压缩范围，保护音频不过度受到影响
  • 保持（Hold）：通过延迟释放来避免强烈的 pumping 效果，用在混响上来避免不自然的声音回弹
  • 预读（Lookahead）：避免快速压缩带来的音头爆裂感，但是会减少压缩带来的音染和特点

# 串联压缩

最主要的目的：动态控制

常见顺序：

  1. 瞬态控制（快启动，快释放）
  2. 动态控制（中启动，快释放）
  3. 动态控制（慢启动，中释放）

# 并联压缩（平行压缩）

最主要的目的：塑造音色，既可以保留素材原本的感觉，又能带来压缩处理的好处

一般用于鼓之类瞬态变化大的声音

# 侧链压缩（Side-Chain）

  • 内部侧链：当前侧链监测信号为所处理信号本身
  • 外部侧链：当前监测信号为其他音轨 (外部) 信号

侧链信号的强弱可以改变压缩的幅度

侧链信号的频段调整可以改变压缩的着重点，越突出的频段，处理幅度越大

# 多段压缩器

针对某一个频段进行压缩，控制频段动态同时起到控制频率的作用

# 限幅器

通过内部削波机制和快速大比例压缩将声音控制在安全范围并提升音量

# 实操

# 底鼓的压缩

中速启动，快速释放：音头出来，音尾放大，音色厚实，丰满

快速启动，快速释放：音色利索，结实

中速启动，慢速释放：听到音头后被掐住不会释放，几乎只剩音头

# 军鼓的压缩

先中速启动，慢速释放

缩短启动时间，塑造合适的音头后，缩短释放时间

# 镲的压缩

快启动时间，偏慢释放：使其连接上

# 通鼓的压缩

稍长启动，短释放

# Bass 的压缩

常规低音基础 bass：中速启动，快释放，大压缩比

细节丰富的演奏型 bass：色染压缩器，平行压缩，LA-3A 光学压缩

# 原声吉他的压缩

背景扫弦：注重声场和节奏感，快启动时间，慢释放时间，较高阈值，较高压缩比（12:1/8:1），串联 LA-3A 进一步控制动态

分解和弦：中速启动，快速释放控制音头压动态；

较短修饰：中速启动，慢释放以延长音尾

# 电吉他的压缩

大失真电吉他：因为动态很小所以基本不需要压缩

clean 电吉他：中启动，快释放

# 人声的压缩

动态过大的音频可以先对低电平部分进行增益处理或使用插件 Vocal Rider 进行自动调整

判断压缩过度：声音达不到预期音量，失去表现力

5-10ms 启动时间，30-50ms 释放

硬朗风格：较硬拐点，高阈值，高比例

温和风格：较软拐点，低阈值，低比例（3:1/4:1）

人声的平行压缩技巧：发送轨挂均衡器提升出色频段，挂载压缩后再挂载一个相同幅度的反向衰减均衡

突出人声细节的技巧：营造近场贴耳感，串联一个压缩器，10ms 启动，50-70ms 释放，内部侧链强调中频

快启动时间，快释放时间：减少声音瑕疵，带出音尾，但会推远声音，

慢启动时间，慢释放时间：强调音头，声音听起来更近，细节保留更多

# 吉他弹唱的压缩

较软拐点，3:1-4:1 的压缩比

0-50ms 启动时间，50-100ms 释放时间

内部侧链减少低频根音对人声压缩的触发

# 钢琴的压缩

软拐点，快启动，快释放，低比例，低阈值，多碰少压，尽量保留

# 总线压缩

作用：小范围控制动态，使得各个音轨更加融合，统一

20-30ms 启动时间，释放自动，1.5/2:1 压缩比，压缩量不超过 2-3dB

通常会内部侧链低切防止底鼓触发压缩

常用压缩器：UAD Neve 33609C，SSL Native Bus Compressor

总线压缩对电平管理的指示意义：总线压缩超过 2-3dB 提示混音音量过大

# 混响器（Reverb）

# 概念

# 声音传播的特点

  • 声音传播的速度 = 0.34 米 / 毫秒
  • 传播过程中高频先衰减
  • 反射面的材质对于频段有重要影响
  • 距离每增加一倍，音量减半

# 混响的定义

混响是一种发生在封闭空间 (如房间) 的声学现象，声源按照球形波全方向传播，声源发出的声音以直线方向到达人耳者或传声器后，由墙面，地面，天花板等所反射的声音再陆续到达，跟随在这些反射声后的是很多界面的多次反射形成的更加密集的反射声。由于声音传播以及反射界面材料吸收都会造成能量衰减，因此反射声的振幅会逐渐减小 。而混响就是由房间界面的反射所形成的全部声音的统称。

# 混响声的两大要素

第一反射：又称早期反射，声音通过空间表面第一时间反射回来的声音，这个声音会直接反映空间的大小尺寸，距离远近（回弹时间）和反射材质（反射频率）。

混响音尾：音尾的时间长短也能反映出房间的大小，反射率等性质

# 混响的种类

# 大厅混响（Hall Reverb）

演奏大厅，歌剧院，教堂，空旷的封闭场所

特点：第一反射不明显，混响时间长，衰减平滑渐进，声音较暗

# 房间混响（Room Reverb）

普通房间，经过声学设计的录音棚，专门录制声音的房间，鼓室等

特点：第一反射迅速，混响时间短

# 混响室混响（Chramber Reverb）

又称密室混响，空无一物的完全封闭房间，墙面为高反射性材料

特点：饱满，延绵，高密度，清晰度高，聚焦感强，占空间多

适用于突出乐器独奏，慢速，委婉的人声

# 弹簧混响（Spring Reverb）

特点：声音有弹性且松散，能听到金属感的高频，根据输入信号的强弱有明显的动态变化，有清晰的音尾

适用于吉他，bass 等

# 板式混响（Plate Reverb）

特点：声音非常紧密，明亮，没有明显的音头音尾区别，容易堆积低频

# 非线性混响（Non Linear Reverb）

特点：非自然的混响声，有被门限卡住的感觉

适用于复古风格的军鼓

# 混响器的参数

  • 混响时间（Decay Time）：混响整体的时间长短，混响消失的时间
  • 空间尺寸（Size）：模拟空间的尺寸大小，和混响时间一定程度成正比。相同的混响时间，尺寸越小，高频衰减越小
  • 预延时（Pre-Delay）：到达人耳的直达声与第一次反射声之间的时间间隔
  • 阻尼（Damping）：混响时间与频率衰减时间的比例。用于调整混响的频率成分随时间的变化。
  • 混合 (Mix)：干湿信号混合比
  • 扩散（Diffusion）：声音的分散状况

# 混响的输入输出模式

Mono in stereo out：无论单声道还是立体声，插件先处理成单声道再计算成立体声

Stereo in stereo out：会保留声像，立体声信号的方位感

# 实操

# 混响的高级控制

  1. 频段调节：利用均衡控制混响亮度（削减高频或低频 / 突出中频 /）Pro R 黄色曲线；混响器后单独挂均衡
  2. 分频段 Decay：让某个频段的混响衰减得更快 Pro 蓝色曲线
  3. 动态控制：
    • 闪避：混响后串联压缩。人声作为外部侧链触发压缩控制混响量，突出人声，填补空隙
    • 门限：类似于 non-linear reverb
    • 压缩： 降低信号动态获得比较平稳的混响，用于打击乐等动态明显的乐器
  4. 调制（Modulatin）：为混响增添色彩

# 鼓的混响

实录鼓很少需要加额外加混响，除非缺少 over head 或 room

使用混响器只对某个部件加混响会破坏鼓的整体感，一般为军鼓，镲，通鼓添加 room 混响而不为底鼓添加

使用适当的闪避避免低频堆积

# Bass 的混响

大多情况下也不需要混响，可以加弹簧混响来增添一些色彩或者混响时间非常短，快速的混响

# 原声吉他的混响

方向 1：润色用，不明显，无预延迟，1s 左右混响时间，早期反射 60ms，小 size（0.1-0.2）

方向 2：营造温暖，宏大，回忆 / 怀旧感，大厅混响

# 电吉他的混响

重型的电吉他一般不需要混响

清音电吉他：板式混响或弹簧混响，基本不加厅堂或房间混响

# 人声的混响

长短混响法：先做一个短混响增加人声宽度，再挂载板式（更亮）/ 大厅混响（低切），一些预延迟，人声作为外部侧链驱动压缩进行闪避

延迟（四分音符，pingpong）串联混响更加具有动感，自然
